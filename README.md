# 🏋️ Apex Forge — Elite Performance Gym

Apex Forge is a premium, high-performance gym website designed with a dark luxury aesthetic. It is built using modern web standards to provide a seamless, interactive experience for athletes and fitness enthusiasts.

## 🔗 Live Preview
Open `index.html` directly in any modern browser. No server required.

## 🚀 Key Features
- **Modern UI/UX**: Dark luxury design with custom cursor effects and scroll reveals.
- **Interactive Performance**: HERO section with canvas particle animations and real-time interactive elements.
- **BMI Calculator**: A professional-grade tool with real-time feedback and health category visualisations.
- **Dynamic Scheduling**: Filterable class timetable for weekly training management.
- **Membership Management**: Tiered pricing system with monthly and annual billing toggles.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.

## 🛠️ Technical Stack
- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom design system using variables, Flexbox, and Grid.
- **Vanilla JavaScript**: All interactive logic built without external frameworks.
- **Canvas API**: High-performance background animations.
- **Intersection Observer API**: Optimized scroll-triggered animations.

## 🔧 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/bmcouma/SoftGrowTech_GymWebsite-ApexForge-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SoftGrowTech_GymWebsite-ApexForge-
   ```
3. Open `index.html` in your preferred browser.
